# API 명세 리뷰

`api_spec.md`는 backend 서브모듈 연결 전의 초안입니다.

## 현재 적합한 점

- frontend API 클라이언트 영역과 backend 리소스가 같은 도메인 단위로 나뉘어 있습니다.
- 인증, 여행방, 장소 반응, 일정, AI, 커뮤니티, 미디어 경계가 분리되어 있습니다.
- AI 응답은 결과뿐 아니라 추천 근거를 포함해야 한다는 기준이 있습니다.

## backend 연결 전 보완

- 권한 모델: 방장, 편집자, 뷰어 역할을 확정합니다.
- 실시간 협업: WebSocket 또는 SSE 이벤트 스키마를 정의합니다.
- 장소 데이터: 외부 API 출처, 캐시, 최신성 정책을 정합니다.
- 미디어: 업로드 제한, 썸네일, 삭제 권한, 신고 정책을 정합니다.
