# 숨길 기능 명세

숨길은 그룹 여행자가 장소 취향을 모으고, 여행방 안에서 경로를 조정하고, 여행 후 기록을 공유하는 협업 여행 설계 서비스입니다.

## 사용자 유형

| 사용자 | 주요 목표 |
| :--- | :--- |
| 여행방 방장 | 여행방 생성, 멤버 초대, 일정 편집, 최종 루트 공유 |
| 여행방 멤버 | 장소 취향 표현, 일정 확인, 댓글/반응 참여 |
| 커뮤니티 탐색자 | 다른 여행자의 후기와 루트 탐색, 저장 |
| 기록 작성자 | 사진과 후기를 정리해 공유 |

## 프론트 화면 범위

| 영역 | Vue 라우트 | 목적 |
| :--- | :--- | :--- |
| 랜딩 | `/` | 서비스 소개와 진입 CTA |
| 인증 | `/login`, `/register` | 로그인/회원가입 |
| 홈 | `/home` | 로그인 후 주요 기능 진입 |
| 내 여행 | `/my-trips` | 여행방 목록과 관리 |
| 취향 수집 | `/trips/:tripId/swipe` | 장소 카드 반응 수집 |
| 경로 설계 | `/trips/:tripId/route` | 지도 기반 일정 편집 |
| 커뮤니티 | `/community`, `/community/feed`, `/community/stories` | 후기와 루트 탐색 |
| 작성 | `/community/story-write` | 여행기 작성 |
| 기록/프로필 | `/record`, `/mypage`, `/settings` | 여행 기록과 사용자 설정 |

## 백엔드 필요 기능

- 인증과 사용자 프로필.
- 여행방, 멤버, 권한.
- 장소 후보, 상세, 선호 반응.
- 일정과 경로 저장.
- AI 경로 초안과 추천 근거.
- 커뮤니티 글, 댓글, 좋아요, 미디어.

## 현재 구현 기준

- 활성 제품은 `frontend/` Vue 앱입니다.
- 백엔드는 아직 planned 상태입니다.
- API 클라이언트는 `frontend/src/api/`에 있으며, 실제 backend 연결 시 이 경계를 기준으로 맞춥니다.
