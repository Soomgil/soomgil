# 캐러셀 가이드

캐러셀과 반복 슬라이더는 Vue 컴포넌트로 관리합니다.

## 기준

- 데이터는 props 또는 store/composable에서 주입합니다.
- 이전/다음 버튼은 실제 `button`을 사용하고 접근 가능한 이름을 둡니다.
- 인디케이터는 현재 위치를 시각적으로만이 아니라 ARIA 상태로도 표현합니다.
- 모바일에서는 스와이프와 버튼 조작이 모두 가능해야 합니다.

## 갱신 기준

- 캐러셀 컴포넌트를 새로 만들면 `.agent/docs/frontend/component_guide.md`와 인벤토리를 갱신합니다.
