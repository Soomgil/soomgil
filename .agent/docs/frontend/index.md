# 프론트엔드 문서

프론트엔드 화면, 라우트, 컴포넌트, 스타일 패턴 문서를 보관합니다. 실제 제품 코드는 `frontend/`에 둡니다.

- `page_map.md`: Vue Router 기준 프론트 화면 맵.
- `component_guide.md`: 컴포넌트 배치와 책임 기준.
- `carousel_guide.md`: 캐러셀 UI 구현 기준.
- `style_effects_guide.md`: 스타일/효과 기준.
- `../process/frontend_team_tdd_checklist.md`: 백엔드 도메인 역할을 기준으로 정한 프론트 담당자, 파일 소유권과 완료 조건.
