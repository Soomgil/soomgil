# Notion 문서화 프롬프트

숨길 프로젝트의 현재 구조를 Notion용 기능/기술 문서로 정리해줘.

## 기준

- 루트는 상위 AI 하네스와 워크스페이스 경계만 관리한다.
- 활성 프론트엔드는 `frontend/` Vue 앱이다.
- 백엔드는 추후 `backend/` 서브모듈로 연결된다.
- 정적 HTML/CSS/JS 목업은 활성 기준이 아니다.

## 포함할 내용

- 서비스 한 줄 소개와 사용자 유형.
- Vue 라우트별 화면 목적.
- frontend API 클라이언트와 backend 필요 endpoint.
- 하네스 명령과 검증 흐름.
- backend 서브모듈 연결 시 갱신할 문서와 검사.
